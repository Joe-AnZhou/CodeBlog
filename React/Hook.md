# React Hook

> *Hook* 是 React 16.8 的新增特性。它可以让你在**不编写 class 的情况下使用 state** 以及其他的 React 特性。

## Hook是什么？

>  Hook是一个特殊的函数，它可以让你`钩入`React的特性，例如`useState`是允许你在React函数组件中添加state的hook。
